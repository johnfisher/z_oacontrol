# Installation Plan for OAC


INITIAL INSTALL
1) start with OA4  vm-oac
) setup networking for the HTTP access and 10.250 access
) Take apache setup file(s), eggs, zenoss deb, xml file, overlay
) Install zenoss, (push into OA4 build)
) run install script on overlay, installing the eggs
) start zenoss, browse to it


UPGRADE INSTALL
Install components manually as directed.
