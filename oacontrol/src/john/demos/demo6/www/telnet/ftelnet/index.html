Instructions

Above are links to the four different methods you can use to run fTelnet and HtmlTerm. Currently fTelnet is more stable and feature-ful than HtmlTerm, so I'd recommend option #3, fTelnet with HtmlTerm fall-back.

Simply view the source to the page with the method you like, and you should see how to implement the client(s) on your site. I've marked the important parts with <!-- IMPORTANT -->. If you have any questions, feel free to drop me a line.

Don't forget to setup Web BBS Assistant, if you aren't running a BBS program that natively supports fTelnet or HtmlTerm

Thanks to TECHWARE BBS for acting as our demo system for both fTelnet and HtmlTerm.

Note: Because of web browser security restrictions, fTelnet may not work correctly when viewed on your local machine. It will work when viewed from a Web server.
